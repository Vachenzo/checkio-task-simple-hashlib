A simple demonstration of the hashlib library, shouldn't take more than 3 minutes to solve...

input and output values

Input: Two string arguments. One is the string to be hashed, another is the hash algorithm.

Output: return the hash as a string of double length, contain only the hexadecimal digits. hint: hexdigest()

Example: 

checkio(b'welcome', 'md5') == '40be4e59b9a2a2b5dffb918c0e86b3d7'
checkio(b'happy spam', 'sha224') == '6e9dc3e01d57f1598c2b40ce59fc3527e698c77b15d0840ae96a8b5e'
